# :information_source: How To Use

Clone the repository

```bash
https://github.com/tavareshenrique/go-restaurant-web.git
```

```bash
# Install the dependencies
yarn
# or
npm install

# Start the JSON Server
json-server server.json -p 3333

# Run GoRestaurant Web
yarn start
# or
npm start
```
